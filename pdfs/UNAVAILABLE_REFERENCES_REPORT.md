## Retrieval Methods Documentation

### Tools Used (artl-mcp)

All retrieval attempts used only artl-mcp tools as required:

1. **download_pdf_from_doi(doi, email)**
   - Uses Unpaywall API to locate open access PDF versions
   - Result: ~5 PDFs downloaded successfully
   - Limitation: Most older papers (pre-2010) have no OA versions

2. **doi_to_pmcid(doi)** + **get_pmcid_text(pmcid)**
   - Converts DOI to PMC ID, then retrieves full text via PMC E-utilities API
   - Result: ~10 papers retrieved as full text
   - Limitation: Only works for papers deposited in PMC

3. **get_full_text_from_doi(doi, email)**
   - Attempts full text retrieval via DOIFetcher
   - Result: Most paywalled papers return 'FULL TEXT NOT AVAILABLE'

4. **pmid_to_doi(pmid)**
   - Converts PMID to DOI for Unpaywall lookup
   - Result: Successful for papers with DOIs; PMID:2455503 (1989) has no DOI

### What Did NOT Work

- **PMC PDF downloads:** PMC blocks automated PDF downloads (403 Forbidden)
- **EuropePMC PDF endpoints:** Return 404 errors
- **Direct HTTP requests:** User correctly rejected this approach in favor of artl-mcp tools only

## Recommendations for Accessing Unavailable References

### Option 1: Institutional Access
If affiliated with a university or research institution:
- Use library proxy/VPN to access publisher websites directly
- Most institutions have subscriptions to major publishers

### Option 2: Interlibrary Loan (ILL)
- Request articles through institutional ILL services
- Usually free for academic users
- Typical turnaround: 1-3 days

### Option 3: Contact Authors
- Many researchers will provide PDFs upon request
- Use ResearchGate, email, or Twitter/X to reach authors

### Option 4: Alternative References

- Task 3 (wider PubMed search) may identify suitable replacements
