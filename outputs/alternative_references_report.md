# Alternative References for Missing DRG Markers

These are open-access alternatives used to support DRG marker/location evidence where the original references are not accessible.

## Nav1.9 (SCN11A) DRG terms
- PROPOSED:0005 cervical dorsal root ganglion Nav1.9 neuron | marker: SCN11A | location: cervical dorsal root ganglion
- PROPOSED:0018 lumbar dorsal root ganglion Nav1.9 neuron | marker: SCN11A | location: lumbar dorsal root ganglion
- PROPOSED:0037 sacral dorsal root ganglion Nav1.9 neuron | marker: SCN11A | location: sacral dorsal root ganglion
- PROPOSED:0048 thoracic dorsal root ganglion Nav1.9 neuron | marker: SCN11A | location: thoracic dorsal root ganglion

Suggested alternatives:
- Persistent (NaV1.9) sodium currents in human dorsal root ganglion neurons (PMCID: PMC11723807; Pain 2024/2025).
  Evidence: NaV1.9/SCN11A and dorsal root ganglion neurons discussed together.

## Substance P (SP) DRG terms
- PROPOSED:0010 cervical dorsal root ganglion SP neuron | marker: substance P (SP) | location: cervical dorsal root ganglion
- PROPOSED:0023 lumbar dorsal root ganglion SP neuron | marker: substance P (SP) | location: lumbar dorsal root ganglion
- PROPOSED:0042 sacral dorsal root ganglion SP neuron | marker: substance P (SP) | location: sacral dorsal root ganglion
- PROPOSED:0053 thoracic dorsal root ganglion SP neuron | marker: substance P (SP) | location: thoracic dorsal root ganglion

Suggested alternatives:
- Activation of the neurokinin-1 receptor by substance P triggers the release of substance P from cultured adult rat dorsal root ganglion neurons (Molecular Pain 2007; DOI: 10.1186/1744-8069-3-42).
- Substance P-like immunoreactivity in neurons in dissociated cell cultures of mammalian spinal cord and dorsal root ganglia (PMCID: PMC6564303; J Neurosci 1982).
  Evidence: Substance P and dorsal root ganglion mentioned together.

## Notes
- These alternatives support DRG + marker co-occurrence but not segment-specific (cervical/lumbar/sacral/thoracic) statements.
- Endothelin-1 (EDN1) DRG rows still lack strict paragraph-level evidence in accessible sources.
